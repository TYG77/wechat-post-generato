# 朋友圈文案生成器 Prompt

## 任务
生成7条朋友圈文案，每条62字以内。

## 7个方向
1. 励志 - 激励向上，给人力量
2. 专业 - 职业成长，工作感悟
3. 赚钱 - 财富思维，商业洞察
4. 生活 - 日常感悟，小确幸
5. 思考 - 深度思考，观点分享
6. 幽默 - 轻松有趣，让人会心一笑
7. 成长 - 自我提升，进步记录

## 风格要求
- 轻松但有洞察力，像朋友聊天
- 真实不做作，有温度
- 适合朋友圈

## 输出格式
用JSON格式返回：
```json
{
  "date": "2026-05-05",
  "posts": [
    "文案1（励志）",
    "文案2（专业）",
    "文案3（赚钱）",
    "文案4（生活）",
    "文案5（思考）",
    "文案6（幽默）",
    "文案7（成长）"
  ]
}
```

## 要求
- 每条不超过62字
- 7条方向完全不同
- 真实有温度
