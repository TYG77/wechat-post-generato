# Lark MCP 认证配置和功能测试指南

## 1. 认证信息配置步骤

### 第一步：获取飞书开发者账号
1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 注册开发者账号
3. 完成企业认证（如果需要）

### 第二步：创建应用
1. 登录开放平台控制台
2. 点击"创建应用"
3. 填写应用基本信息：
   - 应用名称：例如 "AI助手集成"
   - 应用描述：描述应用用途
   - 应用图标：上传应用图标

### 第三步：获取认证信息
1. 在应用详情页找到：
   - **App ID**：应用唯一标识
   - **App Secret**：应用密钥（重要！）

### 第四步：配置应用权限
在"权限管理"中开启以下权限：
- ✅ **云文档**：读取、编辑文档
- ✅ **多维表格**：读取、编辑表格
- ✅ **消息**：发送和接收消息
- ✅ **通讯录**：读取组织架构

### 第五步：更新配置文件
将获取的认证信息更新到以下文件：

#### 更新 `.trae/config.json`：
```json
"lark": {
  "command": "npx",
  "args": ["@larksuiteoapi/lark-mcp"],
  "env": {
    "NODE_OPTIONS": "--no-warnings",
    "LARK_APP_ID": "您的实际AppID",
    "LARK_APP_SECRET": "您的实际AppSecret"
  }
}
```

#### 更新 `.env.lark`：
```env
LARK_APP_ID=您的实际AppID
LARK_APP_SECRET=您的实际AppSecret
```

## 3. 常见问题解决

### 认证失败
**症状**："Invalid app_id or app_secret"
**解决方案**：
1. 检查 App ID 和 App Secret 是否正确
2. 确认应用权限是否开启
3. 检查网络连接

### 权限不足
**症状**："Permission denied"
**解决方案**：
1. 在开放平台开启相应权限
2. 重新获取访问令牌

### 网络问题
**症状**："Network error"
**解决方案**：
1. 检查网络连接
2. 配置代理设置（如果需要）
